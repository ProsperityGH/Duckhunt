# Duckhunt

## About the Game
Welcome to duckhunt!, Shoot as many ducks as you can. Everytime you shoot there is cooldown of half a second. Good luck!

(This was an schoolproject)

## Controls
Left mouse button: shoot (every 500ms there is a cooldown)
