# Duckhunt

### Informatie Game
Welkom bij duckhunt!, Schiet zoveel eenden als je kunt. Elke keer dat je schiet is er een cooldown van een halve seconde. Succes!

### Controls
Linkermuisknop: Schiet (als je schiet kan je voor 500ms niet weer schieten)
